이상 탐지(Anomaly Detection) 개요의 맥락에서, **모델 기반(Model-based) 접근 방식**과 **모델 비의존적(Model-free) 접근 방식**은 이상값(Anomalies)을 식별하는 기본적인 두 가지 전략적 분류를 나타냅니다.

### 1. 모델 기반 접근 방식 (Model-based Approaches)

모델 기반 접근 방식은 데이터의 '정상적인' 행동을 정의하는 수학적 모델을 구축하는 데 중점을 둡니다.

*   **정의:** 모델 기반 접근 방식에서 모델은 **모수적(parametric)**일 수도 있고 **비모수적(non-parametric)**일 수도 있습니다.
*   **이상값 식별:**
    *   이러한 모델에 **잘 맞지 않는** 데이터 포인트가 이상값으로 간주됩니다.
    *   또한, 모델 자체를 **왜곡(distort)**시키는 데이터 포인트도 이상값으로 간주될 수 있습니다.
*   **관련 기술 예시:**
    *   **통계적 접근 방식(Statistical Approaches):** 데이터의 확률 분포 모델(예: 정규 분포)을 가정하고, 해당 모델에 대해 확률이 낮은 객체를 이상값으로 정의합니다. 이는 데이터의 분포(Distribution), 분포의 매개변수(Parameters), 예상되는 이상값의 수(Number of expected outliers)에 의존하는 통계적 테스트를 적용합니다.
    *   **재구성 기반 접근 방식(Reconstruction Based Approaches):** 정상 클래스의 분포에 있는 패턴이 낮은 차원 표현을 사용하여 포착될 수 있다는 가정에 기반하며, 재구성 오차가 큰 객체를 이상값으로 간주합니다. 이는 오토인코더(Auto-encoders)나 주성분 분석(PCA) 등을 사용하여 수행됩니다.

### 2. 모델 비의존적 접근 방식 (Model-free Approaches)

모델 비의존적 접근 방식은 명시적인 확률 분포 모델이나 데이터의 내부 패턴 모델을 구축하지 않고 데이터를 직접 분석합니다.

*   **정의:** 모델 비의존적 접근 방식에서는 **모델을 구축하지 않고** 데이터에서 직접 이상값이 식별됩니다.
*   **기본 가정:** 이 접근 방식들은 일반적으로 데이터에 있는 대부분의 포인트가 정상(normal)이라는 기본 가정을 전제로 합니다.
*   **관련 기술 예시:**
    *   **근접성 기반 접근 방식 (Proximity-based Approaches):** 다른 포인트들로부터 **멀리 떨어진** 포인트(distance to its kth nearest neighbor)를 이상값으로 간주합니다.
    *   **밀도 기반 접근 방식 (Density-Based Approaches):** 객체 주변의 밀도의 역수(inverse of the density around the object)를 이상값 점수(outlier score)로 사용하여 이상값을 식별합니다 (예: Local Outlier Factor, LOF).

이러한 **모델 기반 vs. 모델 비의존적** 분류는 이상 탐지 기술을 이해하고 적용하는 데 핵심적인 틀을 제공합니다.
