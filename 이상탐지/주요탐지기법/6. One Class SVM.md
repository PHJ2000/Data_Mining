**One Class SVM (단일 클래스 서포트 벡터 머신)**은 **주요 이상 탐지 기법(Anomaly Detection Techniques)**의 맥락에서 정상 객체(normal objects)를 분류하는 데 SVM(Support Vector Machine) 접근 방식을 사용하는 특정 기술입니다.

이 기법은 정상 데이터만 사용하여 모델을 구축한다는 점에서 독특하며, 이는 지도 학습(supervised learning)이 아닌 이상 탐지 환경에 적합합니다.

### 1. One Class SVM의 기본 원리

One Class SVM은 다음과 같은 특징을 가집니다:

*   **정상 객체 분류:** 이 기법은 SVM 접근 방식을 사용하여 **정상 객체를 분류**하는 모델을 구축합니다.
*   **단일 클래스 사용:** 이 모델은 제공된 데이터를 사용하여 구축되지만, 이 데이터에는 클래스 레이블(class labels)이 포함되어 있지 않습니다.
*   **이상값 포함 가능성:** 모델 구축에 사용되는 데이터 자체에 이상값이 포함되어 있을 수 있습니다.
*   **분류기 구축의 난제:** 단일 클래스만 주어진 상태에서 어떻게 분류기를 구축할 것인가 하는 문제에 직면합니다.

### 2. 작동 방식 ('원점 트릭')

One Class SVM은 "원점 트릭(origin trick)"을 사용하여 단일 클래스 데이터에서 분류 모델을 구축합니다.

*   **커널 사용:** 일반적으로 **가우시안 커널(Gaussian kernel)**을 사용합니다.
*   **고차원 매핑:** 커널을 통해 모든 데이터 포인트는 단일 초구(unit hypersphere)로 매핑되고, 모든 포인트는 동일한 사분면(orthant, quadrant)에 위치하게 됩니다.
*   **목표:** 이 기법의 목표는 **분리 평면(separating plane)과 원점(origin) 사이의 거리**를 최대화하는 것입니다.
*   **최적화 조건:** 최적화 조건에는 매개변수 $\nu$ (뉴)가 포함되는데, 이 $\nu$는 **이상값의 비율(fraction of outliers)**을 나타냅니다. 분리 초평면으로부터 데이터의 마진은 $\frac{1}{\rho}$에 비례합니다 ($\rho$는 원점으로부터 초평면까지의 거리).

### 3. 장점 및 단점

One Class SVM은 견고한 이론적 배경을 가지고 있지만, 실제 적용에는 몇 가지 어려움이 있습니다.

| 장점 (Strengths) | 단점 (Weaknesses) |
| :--- | :--- |
| **강력한 이론적 기초** (Strong theoretical foundation)를 가지고 있습니다. | **$\nu$의 선택이 어렵습니다** (Choice of $\nu$ is difficult). |
| | **계산 비용이 비쌉니다** (Computationally expensive). |

요약하자면, One Class SVM은 정상 데이터만을 사용하여 고차원 공간에서 원점으로부터 최대한 멀리 떨어져 정상 데이터를 감싸는 경계를 설정함으로써, 이 경계 밖에 있는 데이터를 이상값으로 판별하는 강력한 **모델 기반** 이상 탐지 기술입니다.
