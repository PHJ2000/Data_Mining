통계 기반 접근 방식(Statistical Approaches)은 **이상 탐지 기술(Anomaly Detection Techniques)**의 주요 분류 중 하나이며, 데이터의 **확률 분포 모델**을 기반으로 이상값을 정의하고 식별하는 기법입니다. 이는 데이터의 '정상적인' 행동을 수학적 모델로 정의하는 **모델 기반(Model-based)** 접근 방식의 한 형태입니다.

주요 탐지 기법의 맥락에서 통계 기반 접근 방식의 세부 사항은 다음과 같습니다.

### 1. 정의 및 기본 가정

통계 기반 접근 방식에서 **이상값(Outlier)**은 **데이터의 확률 분포 모델에 대해 낮은 확률**을 갖는 객체로 정의됩니다.

*   **모델 가정:** 일반적으로 데이터 분포를 설명하는 **모수적 모델(parametric model)**, 예를 들어 **정규 분포(normal distribution)**를 가정합니다.
*   **탐지 원리:** 데이터 분포, 분포의 매개변수(예: 평균, 분산), 그리고 예상되는 이상값의 수(신뢰 한계)에 의존하는 통계적 테스트를 적용하여 이상값을 식별합니다.

### 2. 구체적인 통계적 탐지 기법

소스에서는 다양한 통계적 접근 방식이 소개됩니다:

*   **Grubbs’ Test:**
    *   **단변수 데이터(univariate data)**에서 이상값을 탐지하는 데 사용됩니다.
    *   데이터가 정규 분포에서 왔다고 가정합니다.
    *   한 번에 하나의 이상값을 탐지하고, 해당 이상값을 제거한 후 반복하는 방식으로 진행됩니다. 이 테스트는 데이터에 이상값이 없다는 귀무가설($H_0$)과 적어도 하나 이상의 이상값이 있다는 대립가설($H_A$)을 설정합니다.

*   **다변수 정규 분포(Multivariate Normal Distribution) 사용:**
    *   다변수 정규 분포의 확률 밀도 함수를 사용합니다.
    *   **마할라노비스 거리(Mahalanobis distance)**를 계산하며, 이 거리가 통계적 속성이나 영역 지식(domain knowledge)에 기반하여 설정된 임계값($h$)보다 클 경우 해당 객체를 이상값으로 선언합니다.

*   **우도 기반 접근 방식 (Likelihood based approach):**
    *   데이터 집합($D$)이 두 가지 확률 분포, 즉 다수 분포($M$)와 이상 분포($A$)의 혼합 샘플을 포함한다고 가정합니다.
    *   일반적인 접근 방식은 처음에 모든 데이터 포인트가 $M$에 속한다고 가정하고, $M$에 속하는 각 포인트($x_t$)를 $A$로 이동시키면서 **로그 우도(log likelihood)**의 차이($\Delta$)를 계산합니다.
    *   이 차이가 임계값($c$)보다 크면 $x_t$는 이상값으로 선언되고 $M$에서 $A$로 영구히 이동됩니다.
    *   다수 분포 $M$은 나이브 베이즈(naïve Bayes)나 최대 엔트로피(maximum entropy)와 같은 모든 모델링 방법을 기반으로 추정될 수 있으며, 이상 분포 $A$는 초기에 균등 분포(uniform distribution)로 가정됩니다.

### 3. 장점과 단점 (Strengths/Weaknesses)

통계 기반 접근 방식은 명확한 장점과 도전 과제를 가집니다:

| 장점 (Strengths) | 단점 (Weaknesses) |
| :--- | :--- |
| **확고한 수학적 기초**를 가지고 있습니다. | 많은 경우에 **데이터 분포를 알 수 없을** 수 있습니다. |
| 매우 **효율적**일 수 있습니다. | **고차원 데이터**의 경우 진정한 분포를 추정하기 어려울 수 있습니다. |
| **분포가 알려진 경우**에는 좋은 결과를 제공합니다. | 이상값 자체가 분포의 매개변수를 **왜곡**시킬 수 있습니다. |

추가적으로, 데이터 집합의 분포를 식별하는 것(예: 후행이 무거운 분포(Heavy tailed distribution))이나 데이터가 분포의 혼합인지 여부를 판단하는 것도 문제로 작용할 수 있습니다.
