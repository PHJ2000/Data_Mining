클러스터링 기반 접근 방식(Clustering-Based Approaches)은 **이상 탐지 기술(Anomaly Detection Techniques)**의 주요 분류 중 하나로, 클러스터 구조와 데이터 포인트 간의 관계를 활용하여 이상값을 식별합니다. 이 방식은 일반적으로 명시적인 모델을 구축하지 않고 데이터에서 이상값을 식별하는 **모델 비의존적(Model-free) 접근 방식**에 속합니다.

주요 탐지 기법의 맥락에서 클러스터링 기반 접근 방식의 세부 사항은 다음과 같습니다.

### 1. 이상값 정의

클러스터링 기반 접근 방식은 두 가지 주요 방법으로 이상값을 정의합니다:

1.  **클러스터 중심으로부터 멀리 떨어진 포인트:**
    *   **프로토타입 기반 클러스터(Prototype-based clusters)**의 경우, 객체가 **클러스터 중심에서 충분히 가깝지 않다면** 이상값(outlier)으로 간주됩니다.
2.  **작은 클러스터(Small clusters):**
    *   크기가 매우 작은 클러스터 자체를 이상값으로 간주합니다.

또한, 밀도 기반 클러스터(Density-based clusters)의 경우, **밀도가 너무 낮은** 객체는 이상값으로 간주될 수 있으며, 그래프 기반 클러스터(Graph-based clusters)의 경우 **연결이 잘 되어 있지 않은** 객체가 이상값으로 간주될 수 있습니다.

### 2. 이상값의 영향 및 탐지 기법 활용

클러스터링 기반 접근 방식에서 중요한 점은 **이상값이 생성된 클러스터링 결과 자체를 왜곡(distort)할 수 있다**는 것입니다.

이러한 문제에 대응하기 위해, 노이즈(noise)의 개념을 포함하는 클러스터링 기법이 활용되기도 합니다:

*   **DBSCAN (Density-based spatial clustering of applications with noise):**
    *   DBSCAN은 클러스터링 알고리즘이지만, 노이즈(noise) 개념을 통해 이상값을 처리하는 능력을 가집니다.
    *   DBSCAN의 절차는 다음과 같습니다:
        1.  모든 포인트의 $\epsilon$ (엡실론) 이웃을 찾고, `minPts`보다 많은 이웃을 가진 **핵심 포인트(core points)**를 식별합니다.
        2.  이 핵심 포인트들의 연결된 구성요소(connected components)를 찾아 클러스터를 형성하고, 비핵심 포인트(non-core points)를 무시합니다.
        3.  각 비핵심 포인트를 이웃 클러스터에 할당하거나, 그렇지 않으면 **노이즈(noise)**로 할당합니다.
    *   **장점:** DBSCAN은 클러스터 수를 지정할 필요가 없으며, 임의의 모양을 가진 클러스터를 찾을 수 있고, **이상값(노이즈의 개념으로서)에 강건합니다**.

### 3. 장점과 단점

클러스터링 기반 접근 방식은 다음과 같은 장점과 단점을 가집니다:

| 장점 (Strengths) | 단점 (Weaknesses) |
| :--- | :--- |
| **간단합니다** (Simple). | **클러스터링 기술을 결정하기 어려울 수 있습니다**. |
| **많은 클러스터링 기술**을 사용할 수 있습니다. | **클러스터 수를 결정하기 어려울 수 있습니다**. |
| | **이상값이 클러스터를 왜곡할 수 있습니다**. |
| | **노이즈와 이상값을 구별하기 어려울 수 있습니다**. |

DBSCAN의 경우, 경계 포인트(border points)가 둘 이상의 클러스터에 속할 수 있어 완전히 결정론적이지 않으며, **고차원 데이터**에서는 거리 측정(예: 유클리드 거리)이 차원의 저주(curse of dimensionality)로 인해 거의 쓸모가 없어질 수 있습니다. 또한, 밀도 차이가 큰 데이터 세트는 잘 클러스터링하지 못할 수 있습니다.
