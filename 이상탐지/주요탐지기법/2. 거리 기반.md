거리 기반 접근 방식(Proximity/Distance-Based Approaches)은 **이상 탐지 기술(Anomaly Detection Techniques)**의 주요 범주 중 하나이며, 데이터 포인트가 나머지 포인트들로부터 얼마나 멀리 떨어져 있는지를 측정하여 이상값(Anomalies)을 식별하는 기법입니다. 이 방식은 명시적인 수학적 모델을 구축하지 않고 데이터에서 직접 이상값을 식별하는 **모델 비의존적(Model-free)** 접근 방식에 속합니다.

주요 탐지 기법의 맥락에서 거리 기반 접근 방식의 세부 사항은 다음과 같습니다.

### 1. 기본 원리 및 이상값 점수 정의

거리 기반 접근 방식의 핵심은 **"이상값은 다른 지점들로부터 멀리 떨어진 지점"**이라는 가정입니다.

*   **이상값 점수(Outlier Score):** 객체의 이상값 점수는 **$k$번째 근접 이웃(kth nearest neighbor)까지의 거리**로 정의됩니다.
*   **탐지 예시:**
    *   **하나의 근접 이웃(One Nearest Neighbor) 방식**에서는 해당 객체에서 가장 가까운 이웃까지의 거리가 이상값 점수가 됩니다.

### 2. $k$ 선택의 민감성 (Sensitivity to $k$)

거리 기반 접근 방식에서 $k$ 값의 선택은 결과에 매우 민감하게 작용할 수 있습니다.

*   **작은 클러스터 문제:** 만약 클러스터에 있는 데이터 포인트의 수가 $k$보다 작으면, **해당 클러스터 전체가 이상값으로 결정**될 수 있습니다.
*   **밀도 차이 문제:** 밀도가 다른 두 개의 클러스터가 존재할 경우, 한쪽 클러스터($D$)에서는 이상값이 정확히 결정되지만, 다른 클러스터($C$)에서는 이상값이 될 수 있는 포인트가 낮은 이상값 점수를 가질 수도 있습니다. 이는 거리 기반 접근 방식이 **밀도의 변화에 민감**하다는 것을 보여줍니다.

### 3. 장점과 단점

거리 기반 접근 방식은 명확한 장점과 심각한 도전 과제들을 가지고 있습니다:

| 장점 (Strengths) | 단점 (Weaknesses) |
| :--- | :--- |
| **단순합니다** (Simple). | **계산 비용이 비쌉니다** ($O(n^2)$). |
| | **매개변수($k$)에 민감**합니다. |
| | **밀도의 변화에 민감**합니다. |
| | **고차원 공간**에서는 거리 자체가 의미를 잃게 됩니다 (차원의 저주). |

이러한 단점, 특히 밀도 변화에 대한 민감성 때문에, 일부 이상 탐지 기법은 절대적인 거리 대신 **상대적인 밀도**를 고려하는 **밀도 기반 접근 방식(Density-Based Approaches)**으로 발전했습니다 (예: LOF, Local Outlier Factor).
